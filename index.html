Crackz ‚Äî AI Inspection Platform

AI-based computer-vision system for detecting structural defects in harbor infrastructure

Context

Built to solve critical infrastructure inspection challenges for harbor authorities and civil engineering firms. Traditional manual inspection of concrete structures is time-consuming, inconsistent, and expensive.

Architecture

  • Core: Python with PyTorch Lightning for deep learning pipelines
  • Models: U-Net segmentation with 20+ encoder options (ResNet, EfficientNet, MobileNet)
  • Data: Polars for high-performance data processing
  • Deployment: Docker (full/slim/GPU), Kubernetes-ready, Azure Batch for distributed processing
  • Interface: CustomTkinter GUI + Typer CLI with TensorBoard integration

Outcome

Result: Rapid detection workflow reducing inspection time by 70%. Enterprise-ready platform with pixel-perfect crack detection, multi-class segmentation (cracks, corrosion, spalling), and automated reporting with GPS/EXIF support.

WreckGame ‚Äî Unreal-Based Underwater ROV Simulation

Real-time ROV training environment for subsea operations

Context

Developed to provide realistic training for ROV pilots without the cost and logistics of field operations. Combines photogrammetry-captured wreck sites with physics-accurate ROV controls.

Architecture

  • Engine: Unreal Engine with C++ for performance-critical systems
  • Visuals: Photogrammetry pipeline for high-fidelity underwater environments
  • Physics: Custom underwater dynamics and buoyancy simulation
  • Interaction: Real-time lighting and turbidity modeling

Outcome

Result: Immersive training platform enabling cost-effective ROV pilot certification and mission rehearsal in realistic underwater scenarios.
