Crackz — AI Inspection Platform

AI-based computer-vision system for detecting structural defects in harbor infrastructure

Context

Built to solve critical infrastructure inspection challenges for harbor authorities and civil engineering firms. Traditional manual inspection of concrete structures is time-consuming, inconsistent, and expensive.

Architecture

  • Core: Python with PyTorch Lightning for deep learning pipelines
  • Models: U-Net segmentation with 20+ encoder options (ResNet, EfficientNet, MobileNet)
  • Data: Polars for high-performance data processing
  • Deployment: Docker (full/slim/GPU), Kubernetes-ready, Azure Batch for distributed processing
  • Interface: CustomTkinter GUI + Typer CLI with TensorBoard integration

Outcome

Result: Rapid detection workflow reducing inspection time by 70%. Enterprise-ready platform with pixel-perfect crack detection, multi-class segmentation (cracks, corrosion, spalling), and automated reporting with GPS/EXIF support.

WreckGame — Unreal-Based Underwater ROV Simulation

Real-time ROV training environment for subsea operations

meshtop — Meshtastic GPS & Mesh Bridge

Terminal monitoring and APRS/GPS bridge for LoRa mesh radio networks

Context

Built to bridge Meshtastic LoRa mesh radio devices with amateur radio and positioning infrastructure. Enables GPS position forwarding, message relay, and real-time telemetry monitoring — useful for field operations, maritime tracking, and off-grid comms in defense and emergency scenarios.

Architecture

  • Transport: Bluetooth (BLE), USB serial, LoRa/MQTT — switchable at runtime via TUI commands
  • Forwarding: pi-star, APRS-IS, gpsd, and rigtop integration
  • Security: AES-128 and AES-256 per-channel encryption with configurable PSK
  • Interface: Full-screen TUI (true-color) with command history and headless/debug mode
  • Stack: Python, uv, asyncio, TOML configuration

Use Cases

Applications: Off-grid field operations, maritime position tracking, drone telemetry bridging, HAM radio APRS integration, and resilient mesh comms for defense and emergency response.
